# Getting Started - Quick Setup Guide

Welcome to your Sindhi Cultural Society website! This guide will help you customize and manage your website.

## First Steps After Deployment

### 1. Change Admin Password (CRITICAL!)
The website comes with a default admin username and password that **must be changed immediately**:

1. Click the **Admin** button (gear icon) in the top-right corner
2. Login with:
   - Username: `admin`
   - Password: `admin123`
3. Go to the **Settings** tab
4. Change both username and password to something secure
5. Click **Save Settings**
6. Remember your new credentials!

1. In Admin Panel, go to **Leadership** tab

### 4. Add Events
1. Go to **Events** tab in Admin Panel
2. Click **Add New Event**
3. Fill in:
   - Title: Name of your event
   - Description: Details about the event
   - Date: Select from calendar
   - Type: Play, Workshop, Camp, Cultural Program, or Other
   - Venue: Where it's happening
   - Status: Upcoming or Past
   - Image URL: Optional event poster/photo from Imgur
4. Click **Add Event**

### 5. Add YouTube Videos
1. Go to **Videos** tab
2. For each video you want to showcase:
   - Find the video on YouTube
   - Copy the video ID (the part after `watch?v=`)
     - Example: `https://www.youtube.com/watch?v=ABC123` â†’ use `ABC123`
   - Enter video title
   - Paste the video ID
3. Click **Add Video**

### 6. Add Publications (Newspaper Articles)
1. Go to **Publications** tab
2. For each article:
   - Take a photo/screenshot of the newspaper article
   - Upload to Imgur, copy URL
   - Enter article title
   - Enter publication name (e.g., "Times of India")
   - Select publication date
   - Paste image URL
   - Optional: Add external link if article is online
3. Click **Add Publication**

### 7. Add Annual Reports

   - Enter title (e.g., "Annual Report 2024")
   - Enter description
   - Paste file URL
4. Click **Add Report**

### 8. Update Social Media Links
1. In Admin Panel â†’ **Settings** tab
2. Update:

## Language Support

The website automatically shows content in 3 languages:
- **English** (default)
- **Hindi**
- **Sindhi**

Users can switch languages using the language selector in the header.

**Important**: The UI (buttons, labels, navigation) translates automatically. However, content you add (event titles, descriptions) should be added in your preferred language. Consider:
1. Adding events in English with Hindi/Sindhi descriptions
2. Creating separate events for each language
3. Using the language that your audience prefers

## Tips for Best Results

### Photos
- Use square photos (400x400px) for leaders
- Use landscape photos (1200x675px) for events
- Compress large images before uploading to Imgur
- Make sure photos are clear and professional

### Content
- Write clear, concise descriptions
- Use consistent formatting
- Keep titles short (under 60 characters)
- Update regularly to keep content fresh

### Organization
- Use the display order field to arrange leaders by importance
- Mark old events as "Past" to keep upcoming events visible
- Remove outdated content periodically

## Troubleshooting

### Can't login to Admin Panel
- Check if you changed the password and forgot it
- Clear browser cache and try again
- If locked out, contact technical support

### Image doesn't show
- Make sure URL is correct and ends in .jpg or .png
- Test URL by opening in browser directly
- Verify image hosting service allows embedding
- Try re-uploading to Imgur

### Changes not appearing
- Click Save/Update button after making changes
- Refresh the page (F5 or Cmd+R)
- Clear browser cache
- Try a different browser

### YouTube video not playing
- Verify video ID is correct (just the ID, not full URL)
- Check if video is public (not private or unlisted)
- Ensure video is not restricted by owner
